# MEMRISTORES

## Memory Imaging and Structured Radiological Extraction System

MEMRISTORES es un proyecto de investigación orientado a la extracción, estandarización y codificación de biomarcadores neurorradiológicos obtenidos de estudios de neuroimagen en pacientes con enfermedad cerebrovascular.

---

# Objetivo General

Desarrollar un sistema reproducible para transformar informes radiológicos y estudios de neuroimagen en variables estructuradas listas para investigación clínica, inteligencia artificial y análisis estadístico.

---

# Objetivos específicos

- Construir un diccionario neurorradiológico estandarizado.
- Codificar biomarcadores siguiendo STRIVE-2.
- Crear una base de datos anonimizada.
- Generar un sistema reproducible de extracción de variables.
- Facilitar investigaciones en enfermedad de pequeños vasos, ictus y neuroimagen.

- Estenosis intracraneal
- Enfermedad de pequeños vasos

# Estándares utilizados

- STRIVE-2
- MARS
- Fazekas
- ECASS
- Total CSVD Burden Score

---

# Codificación

Cada paciente recibe un código único:

```
ACV-0001
ACV-0002
ACV-0003
```

No se almacenan identificadores personales en este repositorio.
